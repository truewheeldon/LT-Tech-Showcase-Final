1. install python 3.7+
2. clone tech showcase from https://github.com/truewheeldon/LT-Tech-Showcase-Final
3. open terminal, navigate to directory with query.py and requirements.txt
4. either: 'pip install -r requirements.txt'
   or install separately:
        pip install click
        pip install requests
5. input 'python query.py albumid' into the terminal
6. program will ask for a number between 1-50; this will pull up all the photos in the corresponding album
